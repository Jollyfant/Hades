  HADES Changelog

Alpha 0.4.1 2019-04-05
  1. Added downloadable compressed database files on the information page that are supported by HADES. For instructions on how to use the database files refer to the GitHub repository.
Alpha 0.4.0 2019-03-01
  1. Fixed MITP08 showing an incorrectly mapped tomographic model.
  2. Added tomographic models HMSL-S06 and HMSL-P06 (Houser et al., 2008).
  3. Added tomographic models GYPSUM-S and GYPSUM-P (Simmons et al., 2010).
  4. Added tomographic model GAP-P4 (Obayashi et al., 2013 & Fukao and Obayashi, 2013).
Alpha 0.3.1 2019-02-28
  1. Reduced bandwidth usage by factor 10.
Alpha 0.3.0 2019-02-27
  1. Low resolution models sample 1 point per arc-degree and high resolution images 2 points per arc-degree.
  2. Performance improvements for small cross sections.
  3. Committed refactored server & client side code.
  4. Added overview figure showing usage details of the HADES application.
Alpha 0.2.0 2018-04-18
  1. Added SP12RTS-S and SP12RTS-P models (2x2).
  2. Added MITP08 model.
  3. Another attempt at fixing the SVG exporting.
  4. Polar view is now the default.
Alpha 0.1.1 2018-03-06
  1. Added option to show satellite terrain for bathymetry.
  2. Added toggle to show plate boundaries.
  3. SVG exporting for polar charts has been fixed.
Alpha 0.0.0 2017-11-11
  1. Initial release of HADES source code and application.
